  <action>
Create `.claude/skills/print/reference/print-settings.md` with three sections:

**Section 1: Purpose-Based Print Settings**
A lookup table mapping common use cases to full BambuStudio print profiles. Each profile includes ALL of: layer_height, initial_layer_print_height, wall_loops, top_shell_layers, bottom_shell_layers, sparse_infill_density, sparse_infill_pattern, enable_support, print_speed, outer_wall_speed.

Profiles (from research):
- **Decorative / Display**: Fine layers (0.12mm), low infill (10%), slow outer walls for smooth finish
- **Functional / Mechanical**: Standard layers (0.2mm), 3 walls, 30% infill, strong
- **Quick Prototype / Test Fit**: Thick layers (0.28mm), minimal infill (10%), fast (300mm/s)
- **Storage / Container**: Balanced (0.2mm layers, 15% infill, 250mm/s)
- **Flexible / TPU**: Standard layers, 3 walls, slow speed (80mm/s -- critical for TPU)
- **Structural / Load-Bearing**: Standard layers, 4 walls, 40%+ infill, gyroid pattern

For each profile include:
- A "Note" explaining the key tradeoff (e.g., "Fine layers for smooth surface finish")
- BambuLab-specific profile name equivalent (e.g., "Similar to BambuStudio '0.12mm Detail' preset")
- Generic slicer equivalent (e.g., "PrusaSlicer: 0.12mm DETAIL")

**Section 2: Geometry-Based Adjustments**
Modifications applied ON TOP of the purpose baseline:
- Overhangs > 45 degrees: enable_support = 1, support_type = normal(auto)
- Thin walls < 2mm: wall_loops += 1
- Bridge spans > 20mm: note to reduce bridge speed
- Large flat top surfaces: top_shell_layers += 1
- Small detailed features: layer_height = min(layer_height, 0.16)
- Tall thin structures: sparse_infill_density += 10% for stability

**Section 3: BambuStudio INI Parameter Reference**
Complete list of verified Slic3r/BambuStudio parameter names for the config file:
- Quality: layer_height, initial_layer_print_height
- Walls: wall_loops
- Top/Bottom: top_shell_layers, bottom_shell_layers
- Infill: sparse_infill_density, sparse_infill_pattern (grid, triangles, cubic, gyroid, honeycomb, line, rectilinear)
- Support: enable_support, support_type (normal(auto), tree(auto), normal(manual), tree(manual))
- Speed: print_speed, outer_wall_speed, inner_wall_speed, sparse_infill_speed, top_surface_speed
- Temperature: nozzle_temperature, bed_temperature (note: typically set by filament profile, include only if user specifies material)

Include a note that temperature settings are usually controlled by BambuStudio's filament profile and should only be set explicitly if the user has a specific need.

**Add Step 12: Recommend Print Settings and Embed in 3MF**

This step runs AFTER a successful 3MF export (Step 5 or after re-rendering a modified model). It is NOT optional -- always recommend settings when a 3MF is produced.

1. **Determine purpose**: Use the use case from Step 2 clarification (or ask if modifying an existing model without known context). Match to the closest profile in @reference/print-settings.md.

2. **Apply geometry adjustments**: Analyze the .scad source using @reference/printability-checklist.md risk patterns. Apply relevant geometry-based adjustments from @reference/print-settings.md on top of the purpose baseline.

3. **Show recommended settings to user**:
   ```
   Recommended print settings (based on [purpose] + geometry analysis):
     Layer height: 0.2mm
     Walls: 3
     Top/bottom layers: 4/4
     Infill: 30% grid
     Supports: not needed
     Speed: 200mm/s (outer walls: 150mm/s)

   BambuStudio equivalent: Similar to "0.20mm Standard" preset with increased walls
   Generic slicer: PrusaSlicer "0.20mm QUALITY" with 3 perimeters

   These will be embedded in the 3MF file. Review in BambuStudio before printing --
   your saved profiles may override these.
   ```

4. **Inject settings into 3MF** (per user decision: embed in 3MF metadata):
   ```bash
   THREEMF="$OUTPUT_DIR/model.3mf"
   TMPDIR=$(mktemp -d)

   # Unzip the 3MF
   cd "$TMPDIR"
   unzip -q "$THREEMF"

   # Create Metadata directory if needed
   mkdir -p Metadata

   # Write print profile config
   cat > Metadata/print_profile.config << 'PROFILE'
   ; BambuStudio Print Profile
   ; Generated by Claude Print Skill
   ; Purpose: [purpose]

   layer_height = [value]
   initial_layer_print_height = [value]
   wall_loops = [value]
   top_shell_layers = [value]
   bottom_shell_layers = [value]
   sparse_infill_density = [value]%
   sparse_infill_pattern = [value]
   enable_support = [0 or 1]
   support_type = [value]
   print_speed = [value]
   outer_wall_speed = [value]
   PROFILE

   # Repack the 3MF (IMPORTANT: zip from current dir to avoid nested paths)
   rm "$THREEMF"
   zip -q -r "$THREEMF" .
   cd /
   rm -rf "$TMPDIR"
   ```

5. **Caveat to user** (per user decision): Always include: "These settings are embedded as recommendations. BambuStudio may override them with your saved profiles -- review the settings before printing."

6. **If 3MF injection fails**: Fall back gracefully. Show the settings as text in the output summary (Step 7) and note: "I couldn't embed settings in the 3MF file. Here are the recommended settings to apply manually in BambuStudio."

**Update Step 7 (Output Summary)**: Add a line showing the recommended print settings in the summary output, right after the file paths.
