  <action>
    1. Create `backend/app/session/__init__.py` (empty).

    2. Create `backend/app/session/claude_session.py` -- ClaudeSession class:
       - Constructor takes `websocket: WebSocket` reference (mutable -- gets swapped on takeover)
       - `async start()`: Spawn Claude Code subprocess using `ClaudeSDKClient` with `ClaudeAgentOptions`:
         - `system_prompt`: preset "claude_code" with append about 3D printing dashboard
         - `permission_mode`: "acceptEdits"
         - `cwd`: `settings.project_root` (string)
         - `allowed_tools`: ["Bash", "Read", "Write", "Edit", "Glob", "Grep"]
         - Wrap `client.connect()` in `asyncio.wait_for(timeout=settings.claude_connect_timeout)` per Pitfall 5
         - Start idle timer after successful connect
       - `async send_message(text: str)`: Query client and stream response back via WebSocket
         - Use `client.query(text)` then iterate `client.receive_response()`
         - Forward AssistantMessage TextBlocks as "assistant_text"
         - Forward StreamEvent text_delta as "assistant_delta"
         - Forward ResultMessage as "turn_complete" with session_id and cost
         - On exception: call `_recover()`
         - Reset idle timer on each message
       - `async _recover(error: str)`: Disconnect old client, call start() again, send "session_recovered" to WebSocket
       - `_reset_idle_timer()`: Cancel existing timer, create new asyncio task for idle_timeout
       - `async _idle_timeout()`: Sleep idle_warning seconds, send "idle_warning" message, sleep remaining (idle_timeout - idle_warning), send "session_expired", call stop()
       - `async stop()`: Cancel idle timer, disconnect client
       - `async interrupt()`: Call client.interrupt() if client exists
       - IMPORTANT: Cancel idle timer when entering grace period (called from SessionManager.disconnect). Add a `pause_idle_timer()` method.

    3. Create `backend/app/session/manager.py` -- SessionManager class:
       - `active_session: ClaudeSession | None`
       - `_cleanup_task: asyncio.Task | None`
       - `async connect(websocket: WebSocket) -> ClaudeSession`:
         - Cancel any pending cleanup task (reconnect within grace period)
         - If active_session exists: takeover -- swap websocket reference, notify old WS with "session_takeover" (wrapped in try/except per Pitfall 4), close old WS with code 4000
         - Reset idle timer on the reattached session
         - If no active_session: create new ClaudeSession, call start(), set as active
         - Return session
       - `async disconnect(session: ClaudeSession)`:
         - Pause idle timer on the session (Pitfall 3 -- idle timer must not fire during grace period)
         - Start cleanup task with grace_period sleep
       - `async _grace_period_cleanup(session: ClaudeSession)`:
         - Sleep settings.grace_period seconds
         - Call session.stop()
         - Set active_session = None
       - `async shutdown()`: Kill active session immediately (for server shutdown)
       - `async cleanup_orphans()`: Best-effort `pgrep -f claude` check on startup, log warning if orphans found

    4. Create `backend/app/ws/__init__.py` (empty).

    5. Create `backend/app/ws/heartbeat.py`:
       - `async heartbeat(websocket: WebSocket)` coroutine
       - Loop: sleep settings.heartbeat_interval, send PingMessage, catch exceptions to detect dead connection
       - Log heartbeat failures at debug level

    6. Create `backend/app/ws/handler.py`:
       - Import SessionManager, heartbeat, message models, loguru logger
       - `async websocket_endpoint(websocket: WebSocket, session_manager: SessionManager)`:
         - Accept WebSocket
         - Call session_manager.connect(websocket) to get session
         - Send ConnectedMessage with session_id
         - Start heartbeat as concurrent asyncio task
         - Message loop: receive_json, dispatch by "type" field:
           - "chat" -> session.send_message(data["text"])
           - "interrupt" -> session.interrupt()
           - "pong" -> pass (heartbeat response)
           - "idle_response" -> session._reset_idle_timer()
           - unknown -> send ErrorMessage
         - On WebSocketDisconnect: cancel heartbeat task, call session_manager.disconnect(session)
         - On any other exception: log error, cancel heartbeat, try to close websocket

    7. Create `backend/app/main.py`:
       - Configure loguru: remove default, add stdout (INFO), add file sink (settings.log_file, rotation="50 MB", retention="7 days", level=DEBUG)
       - Create logs directory if not exists
       - Create SessionManager instance
       - FastAPI app with lifespan:
         - startup: log "Backend starting up", call session_manager.cleanup_orphans()
         - shutdown: log "Backend shutting down", call session_manager.shutdown()
       - CORS middleware: allow origin http://localhost:5173, credentials=True, all methods/headers
       - GET /api/health: return status "ok" + whether active_session exists
       - WebSocket /ws: delegate to websocket_endpoint(websocket, session_manager)
       - Add `if __name__ == "__main__": uvicorn.run("app.main:app", host=settings.host, port=settings.port, reload=True)`
