  <action>
Test the full pipeline manually (not via skill invocation -- directly execute what the skill would do) to verify each step works on this machine.

**Step 1: Verify OpenSCAD installation**
- Check if `/Applications/OpenSCAD.app/Contents/MacOS/OpenSCAD` exists
- If not, check `which openscad`
- If not found, install via `brew install --cask openscad` (this is the test environment, proceed without asking)
- Confirm with `"$OPENSCAD" --version`

**Step 2: Create test output directory**
- `mkdir -p ~/3d-prints/test-box/`

**Step 3: Generate a test OpenSCAD file**
Write a parametric box with lid to `~/3d-prints/test-box/model.scad` following the patterns from the openscad-guide.md:
- Comment header with description, material (PLA), dimensions
- Parametric variables: width=80, depth=60, height=40, wall=2, corner_r=3, lid_clearance=0.3
- $fn=60
- Separate modules for box body and lid
- Proper CSG with 0.1mm overlap on cuts

**Step 4: Render preview PNG**
```bash
"$OPENSCAD" --imgsize 800,600 --projection p --render --colorscheme DeepOcean --autocenter --viewall -o ~/3d-prints/test-box/preview.png ~/3d-prints/test-box/model.scad 2>~/3d-prints/test-box/openscad.log
```
- Check exit code. If non-zero, read the log, fix the .scad, retry (up to 3 times).

**Step 5: Export 3MF**
```bash
"$OPENSCAD" -o ~/3d-prints/test-box/model.3mf ~/3d-prints/test-box/model.scad 2>>~/3d-prints/test-box/openscad.log
```
- Check exit code. Verify the .3mf file is non-empty (`test -s ~/3d-prints/test-box/model.3mf`).

**Step 6: Verify all outputs exist**
- `model.scad` exists and has parametric variables
- `preview.png` exists and is a valid image (non-zero size)
- `model.3mf` exists and is a valid file (non-zero size, should be a ZIP -- check with `file ~/3d-prints/test-box/model.3mf`)

**If any step fails**: Document the failure, fix the issue, and update SKILL.md with any corrections needed (e.g., different CLI flags, workarounds for macOS headless rendering).
  </action>
  <verify>
    - `test -s ~/3d-prints/test-box/model.scad && echo "SCAD OK"` (exists, non-empty)
